TaskVault

TaskVault is a simple command-line To-Do List application built with Python.

Features

* Add tasks
* View tasks
* Delete tasks
* Easy-to-use menu

Technologies Used

* Python 3

Concepts Practiced

* Variables
* Lists
* User Input
* While Loops
* For Loops
* If-Else Statements
* List Methods (`append()` and `pop()`)
